# writing-notes

我寫 Medium 文章時產生的副產物——實驗對話、研究筆記、草稿素材——都放在這裡。

## 目錄

- **`round-table/`** — 「跨時代思想家圓桌」系列的原始對話。框架改自 [李繼剛（lijigang）老師的 ljg-roundtable](https://github.com/lijigang/ljg-skills/tree/master/skills/ljg-roundtable)，我在他的基礎上做了四件改造：議題錨定真實事件、跨文化邀請、第一人稱穿插、Claude 擔任主持人。每份素材檔都是對應 Medium 文章的原始對話紀錄。

## 說明

這個 repo 裡所有「某某某說」的段落，都是**基於歷史人物思想體系所建構的 AI 思想實驗**，不是真實引言。每份素材檔內都有明確的免責聲明。
